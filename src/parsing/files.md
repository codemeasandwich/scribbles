# Parsing

Input processing and argument parsing.

## Directory Structure

```
parsing/
├── args2keys.js
├── loader.js
└── parceStringVals.js
```

## Files

### `args2keys.js`
Parses arguments and converts them to key-value pairs for logging.

### `loader.js`
Node-hook loader for instrumenting code with scribbles logging.

### `parceStringVals.js`
Template string parsing for log messages with type annotations.
