# Core

Core logging functionality for scribbles.

## Directory Structure

```
core/
├── config.js
├── scribble.js
└── scribblesConfig.js
```

## Files

### `config.js`
Default configuration values for scribbles.

### `scribble.js`
Core scribble logging function that creates structured log entries.

### `scribblesConfig.js`
Scribbles configuration and log level setup.
